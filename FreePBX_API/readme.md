# Configuracion para que les funcione el codigo

## Instalacion de venv

1. Clona el repositorio

   ```bash
   git clone https://github.com/GaelDeMercado/FreePBX_API
   ```

2. Instalar venv en ./

   ```bash
   python -m venv venv
   ```

3. Install requirements.txt

   ```bash
   pip install -r requirements.txt
   ```
